# canephora-genomics-pipeline
Bioinformatics pipeline for SNP calling and phylogenetic analysis in Coffea canephora — developed during M.Sc. dissertation at UFLA (2026). Covers adapter trimming, BWA alignment, GATK 3.8 variant calling, BCFtools/VCFtools filtering, and TASSEL neighbor-joining phylogeny. Portfolio purposes only — data not included.
